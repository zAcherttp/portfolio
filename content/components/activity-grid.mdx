## Usage

Supply column data and render each SVG cell yourself. Pointer geometry and active-cell anchors stay inside the primitive.

```tsx title="activity-grid.tsx" {5}
<ActivityGrid
  columns={weeks}
  cellSize={8}
  gap={2}
  getKey={(item, column, row) => item?.date ?? `${column}-${row}`}
  renderCell={({ item, columnIndex, rowIndex }) => (
    <ActivityCell item={item} x={columnIndex} y={rowIndex} />
  )}
/>
```
