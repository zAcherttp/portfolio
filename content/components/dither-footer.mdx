## Usage

Place the root integration after the footer. `BottomShader` owns only the bottom-of-page scroll nudge; `DitherFooter` owns WebGL checks, theme colors, and its animation lifecycle. Render `DitherFooter` directly when the effect should start automatically.

```tsx title="layout.tsx"
import { Suspense } from "react";
import BottomShader from "@/components/BottomShader";

<>
  <Footer />
  <Suspense fallback={null}>
    <BottomShader />
  </Suspense>
</>
```
