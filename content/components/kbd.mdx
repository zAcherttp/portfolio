## Usage

Static keys do not subscribe to keyboard state. Enable held-state visualization only where it communicates live input.

```tsx title="shortcut.tsx" {4}
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <span>+</span>
  <Kbd keyName="K" reactive>K</Kbd>
</KbdGroup>
```

Use `pressed` when another component owns the state.

```tsx title="controlled.tsx"
<Kbd pressed={recording}>Space</Kbd>
```
